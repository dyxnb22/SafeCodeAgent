# v2.2.2 Notes: MCP Read Tool Loop

## Summary
The bounded agent loop can now call approved read-only MCP tools through the existing registry/adapter/router safety boundary.

## What Changed

**`src/safecode/mcp/loop_executor.py`** (new)
- `MCPLoopResult` — frozen dataclass with observation, success, blocked, exit_code, metadata.
- `MCPReadToolExecutor` — validates via `ToolCallAdapter`, classifies tool, calls `MCPReadOnlyRunner`. Never raises; all failures return a blocked `MCPLoopResult`.
- `_parse_tool_name()` — splits `"server.tool"` into `(server, tool)`. Returns `("", original)` on invalid format.

**`src/safecode/agent/tools.py`**
- `ToolIntentRouter._route_mcp()` — new method: classifies tool name via `classify_mcp_tool()`. Read-only → `mcp.call_readonly` route, `executable_now=True`. Write/unknown → `mcp.propose` route, `executable_now=False`, approval required. Backward-compatible for write/unknown MCP intents.

**`src/safecode/agent/loop.py`**
- `AgentLoop.step()` — branches on `routed.executable_now and routed.route == "mcp.call_readonly"`.
- `AgentLoop._execute_mcp_readonly_step()` — calls `MCPReadToolExecutor.execute()`, records observation, updates session state and journal.

**`src/safecode/state/journal.py`**
- `JournalEventType` extended with `"mcp_call"`.
- `AgentJournalStore.record_mcp_call()` — persists MCP call metadata as a journal event.

## Safety Properties
- Read-only classification via keyword matching in `classify_mcp_tool()` (inherited from `mcp.runner`).
- All calls validated through `ToolCallAdapter` before execution.
- Audit logging handled by `MCPReadOnlyRunner` (unchanged path).
- Executor never raises — all failure paths produce a blocked observation.
- Write/unknown MCP intents still require approval and are not executed by the loop.
- Network, filesystem, and shell policy enforced by existing `MCPReadOnlyRunner`.

## Compatibility
- Existing CLI `sac mcp call-readonly` and `sac mcp propose-write` are unchanged.
- All existing MCP write/unknown intent tests updated to use write-classified tool names.
- Full 799-test suite passes.
