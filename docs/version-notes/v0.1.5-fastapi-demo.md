# v0.1.5 FastAPI Demo

## 实现内容

- `examples/fastapi-demo` 提供完整演示项目。
- MockLLMClient 在 FastAPI demo 中生成 `/health` endpoint patch。
- 完整演示 `ask -> edit -> apply -> history -> rollback`。

## 入口

```bash
cd examples/fastapi-demo
uv run --project ../.. sac edit "给这个 FastAPI 项目添加 /health 接口"
uv run --project ../.. sac apply
uv run --project ../.. sac history
uv run --project ../.. sac rollback --last
```

## 代码走向

```text
FastAPI demo app/main.py
-> ContextCollector 看到 app/main.py
-> MockLLMClient._fastapi_health_patch()
-> PatchParser / Validator / Diff
-> pending patch
-> apply
-> checkpoint
-> rollback
```

## 验收命令

```bash
uv run pytest
cd examples/fastapi-demo
uv run --project ../.. sac edit "给这个 FastAPI 项目添加 /health 接口"
```

## 学习重点

- demo 是产品展示，不是核心架构。
- 先在小项目里证明安全闭环，再接真实 LLM。
