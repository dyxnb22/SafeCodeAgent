# v2.1.4 Release Notes — Context Debug Command

## What's New

**`sac context explain "task"`** — a new read-only debug command that shows why
files and snippets were selected for a given task query.

### Output sections

| Section | Content |
|---------|---------|
| Context Selection | Ranked table of selected files with per-file match score and reason |
| Budget Metadata | Configured `max_bytes` and `max_tokens` from `SafeCodeConfig` |
| Repo Map | File, symbol, import, test, command, and entrypoint counts |

### Options

- `--root PATH` — override project root (defaults to cwd)
- `--limit N` — maximum sources to display (default 10)
