# v0.1.2 Edit Preview

## 实现内容

- `sac edit` 生成 patch proposal。
- `PatchValidator` 校验目标文件存在、路径不越界、SEARCH 唯一匹配。
- `build_unified_diff()` 展示 diff。
- 保存 `.sac/pending_patch.json`。
- 写入 `patch_proposed` audit event。

## 入口

```bash
uv run sac edit "演示一次安全修改"
```

## 代码走向

```text
cli.py edit()
-> AgentOrchestrator.edit()
-> ContextCollector.collect()
-> MockLLMClient.propose_patch()
-> PatchParser.parse()
-> PatchValidator.validate()
-> build_unified_diff()
-> _save_pending_patch()
-> AuditLogger.write()
```

## 验收命令

```bash
uv run sac edit "演示一次安全修改"
uv run pytest tests/test_patch_validator.py tests/test_agent_edit.py
```

## 学习重点

- `edit` 只预览，不写业务文件。
- SEARCH 必须唯一匹配。
- diff 是给人审查的预演。
