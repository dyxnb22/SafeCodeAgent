# v1.8.2 Release Notes — Atomic Sandbox Approval Consumption

## Summary
Closes a TOCTOU race condition in sandbox execution approval consumption.
The approval is now atomically claimed BEFORE `ShellRunner.run()` instead
of after, preventing two concurrent `execute_pending()` calls from both
executing the same approved command.

## Changes

### New: `claim_for_execution()` on `SandboxExecutionApprovalStore`
- Validates all approval binding fields (proposal_id, backend, command_hash,
  preview_hash, project_key, policy_version, expiry, consumed).
- Uses a lock file + `os.replace` for atomic read-validate-write.
- Stale locks (> 60 s) are detected and cleaned up.
- Returns `True` only if the approval was successfully claimed.

### Changed: `execute_pending()` flow
- After preflight passes, calls `claim_for_execution()` BEFORE `ShellRunner.run()`.
- If claim fails, returns blocked result without executing.
- New audit event: `sandbox_execution_approval_claimed` (status=success).
- Old `consume()` call removed from the execute path (the `consume()` method
  remains available on the store for direct use).

### Audit Event
- `sandbox_execution_approval_claimed` written when claim succeeds.
- Contains only safe metadata: proposal_id, backend, purpose, command_head,
  command_hash prefix. Never includes env values or full command text.

## Test Coverage
- 13 new tests in `TestAtomicApprovalClaim`:
  - Concurrent claim: only one succeeds
  - Claim failure does not call subprocess
  - Blocked preflight does not call claim
  - Non-zero exit still consumes approval
  - Legacy approval without `consumed` field claimable once
  - Malformed/expired/mismatched approval: fail closed
  - Audit events: claimed event written, no env leak, failure is blocked
- 3 existing tests updated to match new event naming.
- 50 execution security evals, 155 related sandbox tests pass.

## Compatibility
- Fully backward-compatible with v1.8.0 and v1.8.1 approval files.
- All CLI commands and signatures preserved.
- All existing safety invariants preserved.
