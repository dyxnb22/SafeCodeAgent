# v2.3.1 — Config Wizard

## Summary

Adds `sac setup` for project-level SafeCode configuration.

## Implementation

- `src/safecode/setup.py` writes `.sac/config.toml` with provider, model, network, and safety policy choices.
- The setup flow writes `.sac/setup.env` with `SAFECODE_APPROVAL_DIR` and `SAFECODE_SANDBOX_APPROVAL_DIR` exports for external approval storage.
- Root CLI command `sac setup` supports non-interactive `--yes`, explicit directory options, and `--force`.

## Safety

- Existing setup files are not overwritten unless `--force` is provided.
- Policy and provider values are validated before writing.
