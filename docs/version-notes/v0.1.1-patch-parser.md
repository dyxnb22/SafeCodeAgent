# v0.1.1 Patch Parser

## 实现内容

- `PatchProposal` / `PatchBlock` 数据模型。
- `PatchParser` 解析最小 SEARCH/REPLACE Patch。
- `PatchParseError` 提供清晰格式错误。

## 入口

这一版还没有 CLI 入口，从测试或主类开始读：

```text
tests/test_patch_parser.py
-> PatchParser().parse(...)
-> src/safecode/patch/parser.py
-> src/safecode/patch/models.py
```

## 支持格式

```text
*** Begin Patch
*** Update File: app/main.py
@@
SEARCH:
old text
REPLACE:
new text
*** End Patch
```

## 验收命令

```bash
uv run pytest tests/test_patch_parser.py
```

## 学习重点

- 文本解析先做最小格式。
- 不急着支持 Add/Delete/多文件。
- Patch 文本会变成结构化对象。
