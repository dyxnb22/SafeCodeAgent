# v0.1.0 Ask + Audit

## 实现内容

- CLI 骨架。
- `sac ask` 只读问答。
- `ContextCollector` 读取有限项目上下文。
- `MockLLMClient.ask()` 返回固定回答。
- `AuditLogger` 写入 `.sac/logs/events.jsonl`。

## 入口

```bash
uv run sac ask "这个项目是什么？"
```

## 代码走向

```text
pyproject.toml
-> safecode.cli:main
-> cli.py ask()
-> AgentOrchestrator.ask()
-> ContextCollector.collect()
-> MockLLMClient.ask()
-> AuditLogger.write()
```

## 验收命令

```bash
uv run sac --help
uv run sac ask "这个项目是什么？"
uv run pytest tests/test_audit_log.py
```

## 学习重点

- Typer 命令入口。
- `Path.cwd()` 表示当前运行目录。
- JSONL 适合追加式审计日志。
