# v0.1.4 Rollback + History

## 实现内容

- `sac rollback --last` 恢复最近 checkpoint。
- `sac history` 展示 audit log。
- rollback 后写入 `rollback_completed` event。

## 入口

```bash
uv run sac rollback --last
uv run sac history
```

## 代码走向

```text
cli.py rollback()
-> AgentOrchestrator.rollback_last()
-> CheckpointManager.rollback_last()
-> AuditLogger.write()

cli.py history()
-> AgentOrchestrator.history()
-> AuditLogger.read_recent()
-> Rich Table
```

## 验收命令

```bash
uv run sac rollback --last
uv run sac history
uv run pytest tests/test_checkpoint.py tests/test_audit_log.py
```

## 学习重点

- rollback 依赖 checkpoint metadata。
- history 只读 audit log，不重新推断历史。
- JSONL 日志适合命令行历史展示。
