# v2.1.2: Runtime Consolidation Notes

## Summary

This release consolidates the next product-facing runtime slice before adding
more autonomy. The CLI is split into focused command modules, the interactive
agent loop now uses the structured LLM planning and tool-intent contract, and
context collection now includes repository intelligence plus query-selected
source snippets.

## Implementation Notes

- `src/safecode/cli.py` is now a slim Typer registry that preserves root
  commands and subcommand names.
- Command implementations live in focused `src/safecode/cli_*.py` modules.
- `AgentLoop` uses `LLMClient.plan()` when creating or filling a session plan.
- `AgentLoop` uses `LLMClient.choose_tool()` before routing each step through
  `ToolIntentRouter`.
- `ContextCollector.collect(query=...)` adds compact repo-map summaries and
  selected context snippets while preserving budget metadata.
- Placeholder path and project-detection helpers now fail closed or return a
  concrete coarse label.
- Package metadata and version matrix entries are synchronized to `v2.1.2`.
