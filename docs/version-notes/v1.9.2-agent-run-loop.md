# v1.9.2 Release Notes — Agent Run Loop

## Summary

v1.9.2 adds a bounded run loop for interactive agent sessions:
`sac agent run "goal" --max-steps N`.

## What Changed

- Added `AgentRunResult`.
- Added `AgentLoop.run(...)`.
- Added `sac agent run`.
- The loop stops with one of:
  - `completed`
  - `max_steps_reached`
  - `approval_required`

## Security

- The loop only calls the existing safe `step` primitive.
- It does not apply patches, execute shell commands, call MCP write tools, or
  run sandbox proposals.
- `--max-steps` must be at least 1.

## Backward Compatibility

- Existing `sac agent start/status/clear/step` behavior is unchanged.
