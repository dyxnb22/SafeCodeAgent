# v2.0.4 Release Notes - Demo Workflow Suite

## Summary

v2.0.4 adds a repeatable demo workflow suite for the local coding agent MVP.

## What Changed

- Added `safecode.demo.workflows` with four deterministic workflows:
  FastAPI endpoint, argparse CLI flag, docs-only patch, and failing-test repair.
- Added `sac demo list`, `sac demo show`, and `sac demo materialize`.
- Extended the mock LLM patch generator with stable patches for CLI, docs-only,
  and calculator repair demos in addition to the existing FastAPI patch.
- Added regression tests for workflow coverage, seed project materialization,
  pytest detection, mock patch coverage, and CLI behavior.

## Security

- Demo materialization writes only within the selected workflow directory.
- Existing patch validation, apply checkpoints, shell policy, test detection,
  and audit behavior remain unchanged.
- Demo commands are descriptive; test execution still goes through `sac test`
  and `ShellRunner`.

## Backward Compatibility

- Existing `sac edit`, `sac apply`, `sac test`, and `sac eval` behavior is
  unchanged.
- The existing `examples/fastapi-demo` directory remains available.
