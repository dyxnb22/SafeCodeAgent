# v2.3.2 — IDE Bridge MVP

## Summary

Extends the IDE integration surface from a static manifest to bridge commands for opening pending diffs and selected files.

## Implementation

- `src/safecode/ide/bridge.py` exposes `pending_diff_target()` and `selected_file_targets()`.
- `sac ide open-diff` writes `.sac/ide/pending.diff` and prints its file URI/path.
- `sac ide open-files QUERY` prints safe selected file URI/path rows.
- The IDE manifest now advertises `safecode.openDiff` and `safecode.openSelectedFiles`.

## Safety

- The bridge does not launch local applications by itself.
- Selected files are validated through the filesystem boundary and must be real project files.
