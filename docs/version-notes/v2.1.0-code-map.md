# v2.1.0 Release Notes - Code Map

## Summary

v2.1.0 adds a lightweight repository map for files, symbols, imports, tests,
detected commands, and entrypoints.

## What Changed

- Added `RepoMapBuilder` and `RepoMap` under `safecode.index`.
- Added Python import indexing using `ast`.
- Added test-file detection based on Python test path conventions.
- Added entrypoint detection for `[project.scripts]` in `pyproject.toml` and
  Python `if __name__ == "__main__"` guards.
- Added `sac index map` and `sac index map --json`.
- Added regression tests for repo map contents and CLI output.

## Security

- Repo map uses the existing safe file index from `ContextCollector`.
- Python files are parsed with `ast`; no repository code is imported or
  executed.
- Test command detection remains proposal-only and does not execute commands.

## Backward Compatibility

- Existing `sac index files` and `sac index symbols` commands are unchanged.
- Existing `ProjectTestDetector` behavior is reused, not expanded in this
  version.
