# v2.3.0 — Interactive TUI

## Summary

Adds a compact Rich terminal dashboard for day-to-day agent status.

## Implementation

- `src/safecode/tui/dashboard.py` renders session, plan, pending action, pending diff, and recent journal history.
- `src/safecode/cli_tui.py` adds `sac tui dashboard`.
- `src/safecode/cli.py` registers the new `tui` command group.

## Safety

- Dashboard rendering is read-only.
- Missing sessions, pending patches, and journal files render empty states instead of failing.
