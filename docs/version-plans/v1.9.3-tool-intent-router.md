## Problem
The agent loop needs a typed boundary between model/planner suggestions and
runtime tool execution. Without typed intents, later autonomy would be tempted
to route raw strings directly into tools.

## Goals
- [x] Add `ToolIntent`.
- [x] Add `ToolIntentRouter`.
- [x] Support read, patch, shell, sandbox, MCP, subagent, and report intents.
- [x] Mark patch/shell/sandbox/MCP as approval-required.
- [x] Fail closed for unknown intents or missing required fields.
- [x] Have `AgentLoop.step(...)` record a routed read intent.
