## Problem
Context collection is bounded, but callers need explicit budget metadata to
understand which sources were packed, how much budget was used, and what was
truncated.

## Goals
- [x] Add a reusable context budget packer with byte and approximate token
  budgets.
- [x] Preserve legacy context keys while adding `context_budget` metadata.
- [x] Record packed source keys, source kinds, bytes used, token estimates, and
  truncation notes.
- [x] Truncate UTF-8 text safely on byte boundaries.
- [x] Add ranked selector source metadata while preserving `select(...)`
  compatibility.
