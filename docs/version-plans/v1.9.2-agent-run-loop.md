## Problem
`sac agent step` is safe but manual. The product needs a bounded loop that can
advance several safe steps while still stopping before approval-gated actions.

- [x] Stop on completion, max-step exhaustion, or approval requirement.
- [x] Preserve the one-step safety semantics: no write or execute actions are
